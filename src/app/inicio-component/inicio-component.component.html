Sistema de Reservas para un Restaurante


Generalidades

  • Crea un sistema que permita a los usuarios realizar reservas en un restaurante, seleccionando la fecha, hora, y número de personas.
  • Implementa un panel de administración donde el restaurante pueda ver y gestionar las reservas.
  • El backend debe ser desarrollado en Java con Spring Boot, utilizando JPA/Hibernate para la persistencia de datos.
  • El frontend puede ser desarrollado con React o Angular, con énfasis en la usabilidad y diseño responsivo.
